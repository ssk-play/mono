# GrayBox Console Spec v0.1

## 철학

강력한 제약조건 안에서 창의력을 극대화한다.
제약이 통일된 미학을 만들고, 누구나 게임을 쉽게 만들 수 있게 한다.

## 하드웨어 제약

| 항목 | 스펙 |
|------|------|
| 해상도 | 480×320 논리픽셀 |
| 컬러 | 4단계 그레이스케일 |
| 팔레트 | `#1a1a1a` (BLACK=0), `#6b6b6b` (DARK=1), `#b0b0b0` (LIGHT=2), `#e8e8e8` (WHITE=3) |
| 프레임레이트 | 30fps 고정 |
| 입력 | 방향키(상하좌우) + A버튼 + B버튼 (총 6키) |
| 사운드 | Web Audio 오실레이터 2채널, Square wave only |
| 스프라이트 | 8×8 픽셀, 4색, 개수 제한 없음 |
| 타일맵 | 8×8 셀 기반, 크기 제한 없음 |
| 언어 | TypeScript + 플랫폼 API |
| 에디터 | Monaco 기반 (브라우저 내장, 추후 구현) |

## 컬러 팔레트 상세

```
0 = BLACK  #1a1a1a  ██  배경, 그림자, 투명 대체
1 = DARK   #6b6b6b  ██  중간톤 어두운쪽
2 = LIGHT  #b0b0b0  ██  중간톤 밝은쪽
3 = WHITE  #e8e8e8  ██  하이라이트, 텍스트
```

## 입력 매핑 (키보드)

| 게임 버튼 | 기본 키 | 대체 키 |
|-----------|---------|---------|
| ↑ | ArrowUp | W |
| ↓ | ArrowDown | S |
| ← | ArrowLeft | A |
| → | ArrowRight | D |
| A버튼 | Z | Space |
| B버튼 | X | - |

## 사운드 제약 상세

- 동시 발음 최대 2채널 (ch0, ch1)
- 파형: Square wave만 허용
- 음계 기반 API: "C4", "A#3" 등 음이름+옥타브로 지정
- 용도 분리 권장: ch0=멜로디/효과음, ch1=베이스/서브효과

## 스프라이트 데이터 형식

8×8 그리드, 각 픽셀은 0~3 (컬러 인덱스)
문자열로 정의: 행 구분 없이 64자 또는 \n으로 행 구분

예시 (화살표):
```
00030000
00330000
03330000
33330000
03330000
00330000
00030000
00000000
```

## 디자인 철학

1. **제약은 기능이다** — 제한이 통일된 미학과 낮은 진입장벽을 만든다
2. **AI 바이브코딩 최적화** — TypeScript + 명확한 API로 LLM이 바로 게임을 생성할 수 있어야 한다
3. **코드 안에 문자열 스프라이트** + **비주얼 에디터** 둘 다 지원
4. **Steam 모델** — 만들고, 공유하고, 플레이하는 생태계
