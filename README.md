# GrayBox

> 제약이 곧 창의력이다.

480×320 해상도, 4색 그레이스케일, 2채널 스퀘어 웨이브 — 이 제약 안에서 게임을 만들고, 공유하고, 플레이하는 플랫폼.

## 콘솔 스펙

- **해상도**: 480×320
- **컬러**: 4단계 그레이 (`#1a1a1a`, `#6b6b6b`, `#b0b0b0`, `#e8e8e8`)
- **입력**: 방향키 + A, B (2버튼)
- **사운드**: 2채널 Square wave
- **스프라이트**: 8×8px, 4색
- **언어**: TypeScript

## 시작하기

```bash
# 데모 게임 실행
open demos/star-patrol/index.html
```
